# ShellinG
Basic shell scripting examples and concepts.

## Scripts

| Script | Concept |
|---|---|
| `hello.sh` | User input and echo |
| `var.sh` | Variables and file creation |
| `var3.sh` | Positional parameters (`$1`, `$2`, `$@`, `$#`) |
| `escape.sh` | Escape characters in strings |
| `for.sh` | Basic for loop |
| `for2.sh` | For loop with mixed values and globbing |
| `while.sh` | While loop with user input |
| `case.sh` | Case statement in a loop |
| `ifelse.sh` | If/elif/else conditionals and test expressions |
| `functions.sh` | Functions, local variables, return values, recursion |
| `system_info.sh` | Practical script — system information (macOS) |
| `log_analyzer.sh` | Practical script — log file analysis with grep, awk, sed |
